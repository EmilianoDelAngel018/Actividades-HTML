Ejemplo de Applet con Java

Un applet conforme a lo encontramos en las paginas de Orcle: "es una aplicación hecha en Java que se llama desde un documento HTML y se ejecuta dentro de un navegador." Es necesario recordar que el navegador debe tener instalado el plugin de la Máquina Virtual de Java (JVM).

Máquina Virtual de Java: Es el programa que se encarga de ejecutar el bytecode generado por el compilador Java.

Para entender la forma en que se manejan los applet vamos a ver un ejemplo que presenta el clásico Hola Mundo.

Como puede obserrvarse en la unica variable que utiliza este applet es g que representa un objeto de la clase Graphics sobre el cual vamos a dibujar la cadena de texto que resulta de ejecutar applet.

Para ejecutarlo es neceario mandarlo llamar desde una página HTML utilizando la etiqueta < applet > o la etiqueta < object >. En el primer caso el archivo .class se declara en al atributo code, en el segundo se declara un elemento param donde sus atributos name será code y value será el nombre del applet. Cabe mencionar que en el segundo caso en necesario declarar otros elementos param adicionales y en el elemnto object declarar algunos parámetros como classid, codebase y type. En ambos casos es necesario declarar el tamaño de la ventana del applet con los parámetros widht y height.

La salida del programa será: Hola mundo!!
